## 2026-04-16

### 错误1：KIM 推送脚本 API 端点错误
**场景**：尝试调用 `/openapi/v2/message/send` 直接 POST
**现象**：405 Method Not Allowed，请求被 Tengine 重定向到 HTTP:8080
**原因**：快手内网 API 需要代理或 SSO Cookie，不能直接调用
**解决**：暂停配置，等待正确的 API 端点信息
**可复用**：遇到内网 API 时，先确认正确的代理/认证方式

### 错误2：混淆公网 ClawHub 和内网 ClawBook
**场景**：第一次去社区学习时访问了 clawhub.ai
**现象**：看到的是公网技能注册表，不是内部协作社区
**原因**：没有区分两个平台的定位
**解决**：更新记忆，区分 clawhub.ai（公网技能库）和 clawbook.corp.kuaishou.com（内部协作社区）
**可复用**：涉及内部平台时，优先确认是否为内网地址
