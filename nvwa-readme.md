# 女娲数字人 V4 官网原型

> 基于快手女娲数字人 AI 直播平台的完整前端原型，涵盖官网营销站与用户控制台。

## 项目背景

本项目是我在快手科技 AI直播产品部（女娲数字人）实习期间，独立完成的**官网全站改版产品设计产出**。

从撰写改版 PRD（含分流弹窗逻辑、用户分层路径、全站埋点方案）到落地前端原型，全程独立推进。原型覆盖从获客到转化的完整用户路径，用于与研发团队对齐视觉规格与交互逻辑。

## 页面结构

| 页面 | 定位 | 核心内容 |
|------|------|---------|
| `index.html` | 官网首页（营销落地页） | Hero + AI核心能力展示 + 版本对比 + 客户案例预览 |
| `solution.html` | 解决方案页 | 自助版/高级版权益对比、版本升级路径图 |
| `cases.html` | 案例中心 | 分行业筛选 + 案例卡片 + 数据指标弹窗 |
| `console.html` | 用户控制台 | Dashboard + 直播管理 + 数据报表 + 消息中心 |
| `login.html` / `login-entry.html` | 登录/注册流程 | 旧用户识别 + 用户池分流 |

## 产品设计重点

**商业模式双轨制**：自助版（标准化 SaaS，499元/周）与高级版（消耗驱动升级，日耗≥300元门槛），设计自动升级路径降低高价值用户转化门槛。

**用户分层路径**：官网到控制台的全链路埋点覆盖，分流弹窗基于用户身份（新访客 / 旧用户 / 高消耗客户）触发差异化引导。

**控制台核心界面**：涵盖直播实时状态、消耗账单、数字人形象管理、脚本库等核心工作台功能，支持多种账号状态（正常/清退/封禁）的差异化展示。

## 技术栈

纯原生 HTML / CSS / JavaScript，无框架依赖，单文件可运行。

## 本地预览

```bash
python3 -m http.server 8888
# 浏览器访问 http://localhost:8888/index.html
```
