PHASE 01
Python 读懂代码
🗓 第 1–6 周 · 约 6 周
💡 阶段目标：不要求你会「写代码」，目标是读懂 Claude 生成的 Python 代码。 理解变量、函数、类、API 请求、JSON 处理、条件判断等基础结构。 完成后，当你看到 AI 生成的代码时，不再是「天书」，而是能理解它在干什么、哪里可能出错。

🧭 核心学习原则

🔄
「用中学」优先于「学完再用」
每学完一个概念，立刻找到它在 Claude Code 生成的代码里的对应位置。不要等全部学完再开始动手。
🎯
技术学习 ≠ 背概念，而是建「直觉」
你不需要记住所有公式，你需要的是「看到一个问题，能想到用什么 AI 技术来解决」的直觉。
💬
把 Claude 当学习伙伴
遇到不懂的代码，直接问 Claude 解释。遇到不懂的概念，让 Claude 用类比说明。你已经有最好的老师了。
📤
输出倒逼输入
每两周写一篇学习总结（可以是内部文档）。能用自己的话解释给别人听，才是真的懂了。
