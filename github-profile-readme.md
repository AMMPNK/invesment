## What I'm working on

**At Kuaishou:** Evaluating video generation models (Seedance, Kling, SkyReels), building prompt engineering systems for AIGC live streaming, and designing commercial product architecture. The work is messy and specific in the best way.

**On the side:** I've been building a personal AI workflow harness, mostly inspired by studying Claude Code's source architecture — the agent loop design, tool scheduling, and how it manages context state across long tasks. The practical output is a layered system of Rules, Skills, and a knowledge base running inside my IDE. It handles the high-frequency stuff (PRD drafts, research synthesis, content structuring) so I'm not re-explaining context from scratch every session.

**Longer-term project:** I'm building toward a personal digital twin — an agent that knows my input sources, decision patterns, and output preferences well enough to participate in my work proactively, not just reactively. Right now I've got the input side partly working: I only ingest first-hand material from a small set of researchers I trust, cross-validated, auto-processed into daily and weekly digests. For topics I want to go deeper on, I have it generate a full research report. The output and memory layers are still conceptual, but the direction is clear — genuine human-agent collaboration, not tool use.
