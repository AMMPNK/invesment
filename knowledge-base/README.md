# 白小阳 · 个人知识资产库

> 建立于 2026-04-11 | 核心原则：沉淀判断，而不是记录信息

---

## 这是什么

不是笔记，不是日记。是你的**判断力资产**。

每条记录的价值不在于"你知道某件事"，而在于"你在某个情境下做了某个判断，以及背后的逻辑"。
这些判断积累到一定量，就变成了只属于你的、无法被替代的认知资产。

---

## 怎么触发录入

**场景触发，不靠意志力。** 以下任何一个场景发生时，就录入：

- ✅ 今天在工作中做了一个选择（选A不选B）
- ✅ 发现某个之前的判断是错的
- ✅ 遇到一个问题，发现自己的解法和别人不一样
- ✅ 读到/听到一个让你说"对，就是这个"的观点

**录入不需要完整，三行就够：**

```
情境：[我在做什么]
判断：[我选了什么/做了什么决定]
原因：[为什么，以及我担心什么]
```

## 目录结构

```
knowledge-base/
├── README.md              # 本文件
├── judgments/             # 判断记录（每条一个文件，格式：YYYY-MM-DD-标题.md）
├── patterns/              # 从多条判断里提炼的模式（积累够5条同类判断后提炼）
├── beliefs/               # 认知更新（我原来以为X，现在发现是Y）
└── weekly-digest/         # 每周汇总（每周我来帮你整理）
```
